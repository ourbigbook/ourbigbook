Shorthand link termination character

 Home OurBigBook Markup Macro Link Shorthand link parsing rules
Words: 40
The following characters are the "shorthand link termination characters":
  • space
  • newline \n
  • open or close square bracket [ or ]
  • open or close curly braces { or }
Shorthand internal links and shorthand topic links with a single word terminate if any of these characters are found, see also: Section "Shorthand link parsing rules".
