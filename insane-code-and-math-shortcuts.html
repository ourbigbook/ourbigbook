The insane code and math shortcuts work very analogously and are therefore described together in this section.
The insane inline code syntax: 
a `b c` d
which renders as:
a b c d
and is equivalent to the sane: 
a \c[[b c]] d
The insane block code: 
a

``
b c
``

d
which renders as:
a
b c
d
and is equivalent to the sane: 
a

\C[[
b c
]]

d
