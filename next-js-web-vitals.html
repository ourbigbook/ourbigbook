These are documented at nextjs.org/docs/advanced-features/measuring-performance. To enable them use NEXT_PUBLIC_OURBIGBOOK_LOG_PERF, the front-end-avaible version of OURBIGBOOK_LOG_PERF:
NEXT_PUBLIC_OURBIGBOOK_LOG_PERF=1 npm run dev
If enabled, we see a few metrics printed on the browser console such as:
