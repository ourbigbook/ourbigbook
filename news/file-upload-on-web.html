It is now possible to have create file uploads on OurBigBook Web, e.g.:
  • ourbigbook.com/cirosantilli/_dir
  • ourbigbook.com/cirosantilli/_dir/c
  • ourbigbook.com/cirosantilli/_raw/c/segfault.c
You can upload images, source code, or any other type of file.
Previously, you'd just get broken links when publishing to OurBigBook Web if you linked to your files with the \a macro or other related constructs.
The implementation attempts to follow what already existed for static websites as closely as possible.
The current implementation has a few limitations which should be fixed one day:
  • there is no web UI for file upload, it is exposed via CLI only
  • file previews don't show under corresponding _file articles as they do on static
  • _file articles are not properly auto-generated for files that don's have explicit articles, we just show _raw under _file
