Shorthand topic link (<#Albert Einstein>)

Tags: Shorthand macro syntax Words: 160 Articles: 1
If an shorthand magic link starts with a hash sign (#), then it is converted to a topic link instead of a magic link.
For example:
<#Albert Einstein>
which renders as:
Albert Einstein
is equivalent to:
\x[Albert Einstein]{topic}
