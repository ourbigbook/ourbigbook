OurBigBook Web (Dynamic website)

OurBigBook Project
words: 9k articles: 85
OurBigBook Web is the software program that powers OurBigBook.com, its official flagship instance, see also: Section 1.8. "OurBigBook.com". OurBigBook Web is currently the main tool developed by the OurBigBook Project.
This section contains both end user documentation and developer documentation.
More subjective rationale, motivation and planning aspects of the project are documented at: cirosantilli.com/ourbigbook-com.
